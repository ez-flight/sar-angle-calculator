# Инструкция по использованию

## Быстрый старт

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запуск расчета:**
   ```bash
   python continuous_angle_calc.py
   ```

## Настройка параметров

В файле `continuous_angle_calc.py` можно изменить:

- **NORAD ID спутника** (строка 82):
  ```python
  s_name, tle_1, tle_2 = read_tle_base_file(56756)  # Измените 56756 на нужный ID
  ```

- **Координаты целевой точки** (строка 83):
  ```python
  target_pos = (59.95, 30.316667, 12)  # (широта, долгота, высота в км)
  ```

- **Известные периоды** (строки 88-96):
  ```python
  known_periods = {
      -2.0: [(datetime(2024, 3, 22, 3, 8, 57), ...)],
      # Добавьте свои периоды
  }
  ```

- **Максимальное количество итераций** (строка 122):
  ```python
  max_iterations = 5  # Измените на нужное значение
  ```

## Формат вывода

Скрипт выводит:
- Расчетные углы с точностью до 6 знаков после запятой
- Временные периоды для каждого угла
- Длительность периодов и промежутков
- Общую статистику эффективности использования времени

```

Начальный период: 03:08:57 - 03:09:33

Последовательность углов:
```

## Требования

- Python 3.7+
- Установленные зависимости из `requirements.txt`
- Файл `backup.tle` с TLE данными (или доступ к Space-Track API)
