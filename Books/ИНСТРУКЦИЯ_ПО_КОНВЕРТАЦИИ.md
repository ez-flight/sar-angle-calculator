# Инструкция по конвертации в PDF

## Созданные файлы

1. **TerraSAR-X Staring Spotlight_ru.md** - переведенный текст в формате Markdown
2. **TerraSAR-X Staring Spotlight_ru.html** - переведенный текст в формате HTML

## Способы конвертации HTML в PDF

### Способ 1: Через браузер (рекомендуется)

1. Откройте файл `TerraSAR-X Staring Spotlight_ru.html` в браузере
2. Нажмите `Ctrl+P` (или Файл -> Печать)
3. В диалоге печати выберите "Сохранить как PDF" или "Microsoft Print to PDF"
4. Сохраните файл как `TerraSAR-X Staring Spotlight_ru.pdf`

### Способ 2: Использование командной строки

#### Установка pandoc (если доступны права sudo):
```bash
sudo apt-get install pandoc texlive-latex-base
pandoc "TerraSAR-X Staring Spotlight_ru.md" -o "TerraSAR-X Staring Spotlight_ru.pdf"
```

#### Установка weasyprint:
```bash
pip3 install weasyprint
python3 -c "from weasyprint import HTML; HTML('TerraSAR-X Staring Spotlight_ru.html').write_pdf('TerraSAR-X Staring Spotlight_ru.pdf')"
```

#### Установка wkhtmltopdf:
```bash
sudo apt-get install wkhtmltopdf
wkhtmltopdf "TerraSAR-X Staring Spotlight_ru.html" "TerraSAR-X Staring Spotlight_ru.pdf"
```

### Способ 3: Использование онлайн конвертеров

1. Загрузите файл `TerraSAR-X Staring Spotlight_ru.html` на один из онлайн-сервисов:
   - https://www.ilovepdf.com/html-to-pdf
   - https://www.freepdfconvert.com/html-to-pdf
   - https://html2pdf.com/

2. Скачайте полученный PDF файл

## Примечания

- Markdown файл можно редактировать в любом текстовом редакторе
- HTML файл можно открыть в любом браузере для просмотра
- PDF файл будет создан после конвертации одним из указанных способов
